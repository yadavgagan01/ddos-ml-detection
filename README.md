# DDoS Detection using ML & DL

## Models Used
- Logistic Regression
- SVM
- Naive Bayes
- AdaBoost

## Dataset
CIC-DDoS2019

## Steps
1. Data preprocessing
2. PCA (Dimensionality Reduction)
3. Model training
4. Evaluation (Accuracy, F1, ROC)

Run main.ipynb in Jupyter Notebook
